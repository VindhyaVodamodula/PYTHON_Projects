# Menu Calculator (Python)

This project is a command-line calculator built in Python that allows users to perform different mathematical operations through a menu-driven interface.

The program continuously runs in a loop so the user can select different operations without restarting the program.

## Features

- Menu-driven interface
- Perform multiple mathematical operations:
  - Addition
  - Multiplication
  - Subtraction
  - Division
  - Exponent
  - Remainder
- Input validation for numeric values
- Error handling using try/except
- Continuous program loop allowing multiple calculations

## Technologies Used

- Python
- While loops
- Conditional statements
- Input validation
- Exception handling (try/except)
