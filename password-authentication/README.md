# Password Authentication Program

This project is a simple Python command-line program that demonstrates the use of loops and conditional statements to validate user input.

The program asks the user to enter a password. If the password is correct, the user is welcomed and asked to enter their name. The program then prints a customized response depending on the name entered.

## Features

- Password verification using conditional statements
- Repeated user attempts using a loop
- Input validation with retry option
- Conditional responses based on user input
- Interactive command-line interface

## Technologies Used

- Python
- While loops
- If / elif / else conditional statements
- User input handling

## Program Logic

1. Ask the user to enter a password.
2. If the password is incorrect, show an error message.
3. Ask the user if they want to try again.
4. If the password is correct, welcome the user.
5. Ask the user for their name.
6. Display a different message depending on the name entered.
