Project: To-Do List Application

This project is a command-line To-Do List application built using Python.  
It allows users to create and manage tasks with deadlines, priorities, and notes.

### Features
- Add tasks with name, deadline, priority, and notes
- Automatically record task creation date
- Validate date format using Python datetime module
- Organize tasks using lists and dictionaries
- Display tasks in a structured format

### Technologies Used
- Python
- datetime module
- Functions
- Lists and dictionaries
- Loops and conditional logic

### What I Learned
- Designing a command-line application
- Handling user input and validation
- Working with Python data structures
- Managing dates with the datetime module
- Structuring programs using functions
