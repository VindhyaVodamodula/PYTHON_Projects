# Geometry Calculator (Python)

This project is a menu-driven Python program that calculates the area and perimeter of basic geometric shapes. The program demonstrates the use of user-defined functions, loops, and input validation.

The application allows the user to select a geometric operation from a menu and then enter the required values for calculation.

## Features

- Menu-driven program structure
- Separate functions for each calculation
- Calculates:
  - Area of square
  - Area of rectangle
  - Area of circle
  - Perimeter of square
  - Perimeter of rectangle
  - Perimeter of circle
- Input validation using try/except
- Continuous execution until the user exits the program

## Technologies Used

- Python
- Functions
- Loops
- Conditional statements
- Exception handling
- Mathematical operations
