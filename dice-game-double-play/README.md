# Dice Game: Double Play (Python)

This project is an interactive command-line dice game built in Python. The game allows a user to place a wager, guess the outcome of a dice roll, and win or lose based on the result. It also includes a second-round "double play" option that increases both the risk and the possible reward.

This project is meaningful to me because it reflects my growing confidence in turning programming concepts into something interactive, engaging, and personal. It helped me move beyond simple exercises and think more about user experience, game flow, and outcome tracking.

## Features

- Random dice roll generation using Python
- User choice to start or exit the game
- Wager system with winning and losing conditions
- Number guessing logic
- Optional second round with increased stakes
- Tracking of winnings across rounds
- Interactive command-line gameplay

## Technologies Used

- Python
- `random` module
- Conditional statements
- User input handling
- Arithmetic operations
- Basic game logic

- working with conditional statements for different game outcomes
- handling user choices and wagers
- calculating winnings and losses across rounds
- building a more engaging command-line program
